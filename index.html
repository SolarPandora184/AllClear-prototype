Toggle which feeds appear on the map. Sources marked NEEDS KEY require you to paste an API key in the CONFIG section of the code. See the README notes at the bottom of index.html.

PulsePoint Live (official widget)

This is PulsePoint's own official embed - a live, view-only widget. Its incidents can't be merged into the map above (browser security prevents any site from reading another site's iframe contents), so it's shown here on its own.

Account

Creating an account lets your preferred sources sync across devices. Firebase must have Email/Password sign-in enabled in your project (Firebase console -> Authentication -> Sign-in method) for this to work.
